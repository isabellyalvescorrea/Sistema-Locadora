# Sistema básico - locadora

Desenvolvido em Javascript para rodar no terminal,
permite a uma locadora:

- Cadastrar filmes
- Excluir filmes
- Cadastrar clientes
- Remover clientes
- Mostrar informações (clientes e filmes cadastrados)
- Agendar clientes para um filme

# Tecnologias utilizadas:
- Javascript
- Vs code
